API Beschreibung

Die Moviemate API ist speziell fuer Kinos entwickelt und hilft dabei, den Besuchern personalisierte Filmauswahlen zu praesentieren. Basierend auf den zuvor eingegebenen Daten und Praeferenzen der Benutzer, schlaegt die API Filme vor, die dem individuellen Geschmack der Besucher entsprechen. Diese API kann nahtlos in Kino-Websites integriert werden und verbessert das Benutzererlebnis durch personalisierte Empfehlungen.

GET /api/data
Host: moviemate.mabu2807.de
Content-Type: application/json

Beispie Request-Body

body: { "movies": [ { "movieId": '' // ID des Films als Integer oder UUID, "title": "Muster Titel", "genres": "Adventure, Animation, ..." // Genres als String getrennt durch Komma, "description": "Movie description" // Beschreibung des Films als String, }, ], "user":{ "genres": ["Adventure", "Animation", "Children", "Comedy", "Fantasy", ...] // Genres als Array von Strings, "ratings": [ {"movieTitle": 1, "rating": 5}, ... ] // Bewertungen als Array von Objekten mit movieTitle und rating } }

Bespiel Response

{ movies: [ {"movieId": 1 // integer oder UUID, "score": 19 // Integer}, {"movieId": 2, "score": 85}, {"movieId": 3, "score": 97}, {"movieId": 4, "score": 5}, {"movieId": 5, "score": 50}, {"movieId": 6, "score": 90}, ] }

Hinweise

  • Die Daten der User muessen selbst erhoben werden!
  • Um so mehr Daten uebermittelt werden, umso genauer ist die Vohersage!
